Adds social media accounts to Ultimate Member profile

== Description ==
Great easy to use plugin to add your social media accounts to your Ultimate Member profile.

= Available Social Networks=

- Facebook
- Twitter
- Instagram
- LinkedIn
- Xing
- Renren
- Google+
- Whatsapp
- Tumblr
- Pinterest
- Youtube
- VK
- Vine
- Website

== Installation ==
1. Download and extract the zip archive
2. Upload `social-links-for-ultimate-member` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu in WordPress
